Getting Started with Mia

Mia is a utilitarian medical imaging software. It provides a set of tools and filters that allow you to load and annotate images. It offers a simple interface that results in suprisingly powerful utility.

Mia includes many of the tools and filters found in flashphoto, so, for an in-depth explanation of the tools and filters found in both applications please follow the link at the bottom of the page to the flashphoto guide.
