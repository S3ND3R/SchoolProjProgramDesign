The Imagetools API

What is it?

The Imagetools API offers a versitile set of tools for creating diverse image editing applications. The imagetools API supports digital painting and image editing applications. It comes with two fully functional applications that demonstrate the power and range of expression that this API is capable of.

How do I use it?

A tool for both users and developers. Imagetools applications are intuitive and quick to pick up and operate. Users looking to explore getting started with either mia or flashphoto can explore the user tutorials below. Developers looking to harness the power of the imagetools API for their own applications or expand on its functionality can look into the Developing with Imagetolls link.
