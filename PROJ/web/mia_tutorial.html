Loading an Image to Sharpen and Annotate

Let's try loading in an image, then sharpening the image to increase visibility. Then we will annotate the image as if we have found a problem. Finally we will save the image for future access.

Open a fresh Mia application.
Hit open and navigate to the file you wish to work on.
Now that the image is loaded we will apply sharpen.
Apply sharpen to the image.
Once the image has been sharpened hit Mark, which now lets you stamp areas of interest.
After you have marked an area of interest, select the pen to annotate.
If the image is ready, hit save to store the image in a file.
You have now saved the image for later, great job!
