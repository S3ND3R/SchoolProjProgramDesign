Flashphoto Tools

Tools allow you to directly apply paint to the canvas. Flashphoto has six seperate tools for painting on the canvas along with an eraser tool. Each tool provides its own unique brush stroke when applied to the canvas. All tools can easily be selected from the menu.

The 7 design tools are:

  • Pen Tool
    • Designed to be like a pen. It completely covers any area of the canvas with the color it has been set to.
  • Calligraphy Pen Tool
    • Designed almost exactly like the pen, except that to simulate a calligraphy pen the brush stroke is tilted.
  • Chalk Tool
    • Designed to feel like a piece of chalk. The tool does not completely cover the canvas, instead it randomly leaves some area exposed.
  • Highlighter Tool
    • Designed to represent a highlighter. The tool has a high level of transparancy, so the colors on the canvas underneath are visible.
  • Spray Can Tool
    • Designed to work like a spray can. The tool gradually applies paint to the canvas if it is held in place. The amount of paint applied decreases the farther you are from the center.
  • Blur Tool
    • This special tool blurs the colors that are already on the canvas. It has the feel of smudging the paint on the canvas with your thumb.
  • Eraser Tool
    • This tool functions like an eraser. It removes all paint that has been applied to the canvas, and returns it to the original background color.

Color and tool size Selection

The menu in flashphoto has options for selecting both the size and color of the tools. Tool color can be selected from a simple color graphic. Tool size can be adjusted using the Tool Size slider.
