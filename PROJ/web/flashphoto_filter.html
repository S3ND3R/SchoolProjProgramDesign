Flashphoto Filters

Filters allow you to apply changes to the entire canvas. Flashphoto has eight different Filters that can be applied to the canvas. Each filter can be adjusted to create powerful visual effects on the canvas. Like tools, filters can easily be selected from the menu.

The 8 filters are:

  • Blur
    • The blur filter will blur an image by the amount specified by the slider. The larger the value the blurrier the image
  • Motion Blur
    • Designed almost exactly like the blur filter, except that blur is applied in the direction selected.
  • Sharpen
    • The sharpen filter brings out the edges of the shapes in the image by the amount selected by the slider. A higher value leads to greater accentuation.
  • Edge Detect
    • The edge detect filter removes everything but the edges in the image.
  • Threshold
    • The threshold filter will reduce all colors in the image to either balck or white. It chooses whether to make a color black or white based on whether the color value of a pixel is greter than the value set by the threshold slider.
  • Saturate
    • The saturate filter increases or decreases the colorfulness of the image based on the value set by the saturation slider.
  • Adjust RGB
    • The filter for adjusting RGB lets you change the individual color values in the image. The values will increase or decrease in intensity depending on the value set in the slider
  • Quantize
  • The quantize filter will reduce the number of colors in the image based on the number of bins set in the slider. The lower you set the number of bins the closer the color will move towards its primary value.
